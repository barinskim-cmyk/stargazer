# Созвездие (Stargazer)

Мобильная игра-головоломка: соединяй точки, замыкай многоугольники, зарабатывай очки. Обёрнута в нативное iOS-приложение через Capacitor.

---

## Стек технологий

| Слой | Технология |
|------|-----------|
| Фронтенд | HTML5 Canvas, vanilla JS, CSS custom properties |
| Упаковка | [Capacitor v8](https://capacitorjs.com/) → iOS WKWebView |
| Шрифты | Google Fonts: Raleway (UI), Caveat (chalk-тема) |
| Хранилище | `localStorage` (прогресс, настройки) |
| Бэкенд | **Отсутствует** — игра полностью офлайн |

> **Что не хватает на бэкенде:** сейчас нет сервера, API, базы данных или синхронизации. Весь прогресс хранится локально. Для будущего: потребуется API для сохранения имён созвездий, лидерборда, мультиплеера.

## Архитектура фронтенда

Весь код — один файл `www/index.html`. Структура внутри:

```
<head>
  <style>           CSS: темы, экраны, оверлеи
</head>
<body>
  #start            Экран меню
  #journey          Экран «Мой путь»
  #game             Игровой экран
    #top-panel      Счёт, имена, ход
    #canvas-area    Игровой canvas + оверлеи
    #bottom-panel   Скрытая панель (ранг, тема, последний ход)
  #vignette-overlay Глобальная виньетка (CSS)
  #grain-overlay    Зернистость (CSS)
  <script>          Весь JS во внутренней IIFE
```

### Поток выполнения

```
Клик по canvas
  → canvas.click
    → validate(a, b)
      → applyMove(a, b)
        → checkScore(a, b, player)
        → updateUI()
        → draw()
        → isGameOver()
          → endGame()
            → progressionUpdate()
            → startRankUpCeremony() ──→ finishEndGame()
            → finishEndGame()         (без ранг-апа)
```

| Переменная | Назначение |
|-----------|-----------|
| `--th-bg` | Фон экрана |

| `--th-text` / `--th-text2` | Основной / вторичный цвет текста |
| `--th-btn-bg` / `--th-btn-text` / `--th-btn-border` | Кнопки |
| `--th-accent` | Акцентный цвет (линии, выделение) |
| `--th-btn-glow` | Цвет свечения кнопок при hover |
| `--th-vignette` | Интенсивность виньетки (0–0.45) |
| `--th-font` | Семейство шрифтов |

## Правила игры (валидация)

Ход (a → b) недопустим если:
1. Ребро уже нарисовано
2. Другая точка лежит на отрезке (a,b)
3. Отрезок пересекает существующее ребро
4. Отрезок проходит через захваченную область
5. Ребро замыкает только фигуры без свободных точек внутри (невалидная фигура)

Очко начисляется за наименьший валидный замкнутый многоугольник.

## Конечные автоматы: экраны завершения игры

### Проигрыш (AI выиграл)
```
endGame()
  → finishEndGame()
    → clearRect (canvas пуст)
    → canvas-area.classList.add('lost')
    → #loss-content visible
```

### Ничья
```
finishEndGame()
  → draw() (созвездие отрисовано)
  → canvas-fade-overlay opacity 0→1 (0.8s)
  → #victory-content "Ничья" opacity 0→1 (0.6s)
```

### Победа
```
finishEndGame()
  → draw()
  → canvas-fade-overlay + #c scale(0.42) translateY(-50%) (одновременно, 1.2s)
  → t=1000ms: canvas-fade-overlay opacity→0 (созвездие светлеет маленькое)
  → t=1600ms: #name-constellation opacity→1
  → пользователь вводит имя → сохранение → startGame()
```
