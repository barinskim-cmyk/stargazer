# Созвездие (Stargazer)

Мобильная игра-головоломка: соединяй точки, замыкай многоугольники, зарабатывай очки. Обёрнута в нативное iOS-приложение через Capacitor.

---

## Стек технологий

| Слой | Технология |
|------|-----------|
| Фронтенд | HTML5 Canvas, vanilla JS, CSS custom properties |
| Упаковка | [Capacitor v8](https://capacitorjs.com/) → iOS WKWebView |
| Шрифты | Google Fonts: Raleway (UI), Caveat (chalk-тема) |
| Хранилище | `localStorage` (прогресс, настройки, мои созвездия) |
| Бэкенд | **Supabase** — публичные созвездия в общем небе, email-авторизация |

## Архитектура фронтенда

Весь код — один файл `www/index.html`. Структура внутри:

```
<head>
  <style>           CSS: темы, экраны, оверлеи
</head>
<body>
  #auth-screen      Авторизация (email/password или гость)
  #tutorial-screen  Интерактивный туториал (первый запуск)
  #home-screen      Главный экран
  #start            Выбор режима игры
  #journey          Мои созвездия + достижения
  #sky-hub-screen   Хаб: «Мои созвездия» / «Небо»
  #sky-screen       Общее / личное небо (галактика)
  #settings-screen  Настройки
  #game             Игровой экран
    #top-panel      Счёт, имена, ход
    #canvas-area    Игровой canvas + оверлеи победы
    #bottom-panel   Ранг, последний ход, тема
  <script>          Весь JS (глобальный скоп)
```

### Поток выполнения

```
Клик по canvas
  → canvas.click
    → validate(a, b)
      → applyMove(a, b)
        → checkScore(a, b, player)
        → updateUI()
        → draw()
        → isGameOver()
          → endGame()
            → progressionUpdate()
            → startRankUpCeremony() ──→ finishEndGame()
            → finishEndGame()         (без ранг-апа)
```

## Авторизация

| Путь | Поведение |
|------|-----------|
| Email / Password | Supabase Auth, `sb.auth.signInWithPassword()` / `signUp()` |
| «Продолжить без входа» | `guestMode = 'true'`, `userId = 'null'` |
| Повторный запуск | `sb.auth.getSession()` восстанавливает сессию автоматически |
| Выход | `sb.auth.signOut()` + очистка localStorage |
| Туториал | Не запускается если `rating > 0` или `tutorialDone` установлен |

## Победный экран — логика сохранения

| Пользователь | Кнопка | Действие |
|---|---|---|
| Авторизован | «Сохранить в небо ✦» | localStorage + Supabase, с `publishedAt` |
| Гость | «Сохранить в мои созвездия» | Только localStorage, без `publishedAt` |
| Туториал (первая игра) | — (кнопок нет) | Автосохранение в localStorage в конце туториала |

## CSS: система тем

Темы задаются через CSS custom properties, активируются через `applyTheme(name)`.

| Переменная | Назначение |
|-----------|-----------|
| `--th-bg` | Фон экрана |
| `--th-panel-bg` | Фон панелей |
| `--th-text` / `--th-text2` | Основной / вторичный цвет текста |
| `--th-btn-bg` / `--th-btn-text` / `--th-btn-border` | Кнопки |
| `--th-accent` | Акцентный цвет (линии, выделение) |
| `--th-btn-glow` | Цвет свечения кнопок при hover |
| `--th-vignette` | Интенсивность виньетки (0–0.45) |
| `--th-font` | Семейство шрифтов |

### Доступные темы

| Тема | Фон | Акцент | Особенности |
|------|-----|--------|-------------|
| `stargazer` | `#0b0b14` тёмно-синий | `#C09358` | Звёзды-спрайты, мерцание, пульсация |
| `minimal` | `#FFFFFF` белый | `#000000` | S-кривые на экране победы |

| `chalk` | `#F9F8F6` бумага | `(rainbow)` | Мелковая текстура, спирали, дрожащие линии |

## Правила игры (валидация)

Ход (a → b) недопустим если:
1. Ребро уже нарисовано
2. Другая точка лежит на отрезке (a,b)
3. Отрезок пересекает существующее ребро
4. Отрезок проходит через захваченную область
5. Ребро замыкает только фигуры без свободных точек внутри

Очко начисляется за наименьший валидный замкнутый многоугольник.

## AI: уровни сложности

| Уровень | Название | Поведение |
|---------|----------|----------|
| 1 | Начинающий | 90% случайных ходов |
| 2 | Любопытный | 70% случайных |
| 3 | Опытный | 50% умных |
| 4 | Мудрый | 20% случайных |
| 5 | Главный Звездочёт | Всегда умный |

Умный ход: очко > не давать очко > цепочки > угрозы.

## Прогрессия

### Ранги

| Ранг | Очки | Название |
|------|------|---------|
| 1 | 0 | Наблюдатель |
| 2 | 300 | Картограф |
| 3 | 800 | Следопыт |
| 4 | 1800 | Хранитель |
| 5 | 3500 | Звездочёт |

### Начисление рейтинга
- Победа: рейтинг-очки рассчитываются по `computeGamePoints()`, делятся на 10, округляются вверх
- Поражение: −10 очков рейтинга
- Ничья: очки не начисляются

### Достижения (8 штук)

| ID | Условие |
|----|---------|
| `first_win` | Первая победа |
| `score_10` | 10 очков в одной партии |
| `win_3_streak` | 3 победы подряд |
| `win_10` | 10 побед всего |
| `all_themes` | Сыграть всеми темами |
| `long_game` | Партия на 21 точку |
| `rank_up` | Получить повышение ранга |
| `tutorial` | Пройти туториал |

## iOS-специфика

- **Capacitor 8.1.0**, `contentInset: "never"` — отключает автоматические WKWebView insets (фикс iOS 26 layout shift)
- **`body { position: fixed }`** — блокирует layout на viewport, предотвращает смещение
- **Ручная синхронизация**: `www/index.html` → `ios/App/App/public/index.html` (без `cap sync`)
